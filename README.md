# Lab 3 — Web Applications

This workspace includes two small web applications built with HTML, CSS and JavaScript:

- `todo/` — Simple Todo List application (add, complete, delete, persistence via localStorage)
- `password-generator/` — Secure Password Generator (configurable length and character sets)
- `breakout/` — optional placeholder for Breakout game

Quick steps to initialize a local git repo and push to GitHub:

```bash
cd lab3_shakirov
git init
git add .
git commit -m "Initial web apps: todo and password-generator"
# Create a new repo on GitHub (via website) and follow instructions, e.g.:
# git remote add origin https://github.com/USERNAME/REPO.git
# git branch -M main
# git push -u origin main
```

Open `todo/index.html` and `password-generator/index.html` in a browser to run the apps.
